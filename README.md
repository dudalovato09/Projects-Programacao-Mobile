# Programação Mobile

Este repositório contém os exercícios e projetos desenvolvidos na disciplina de Programação Mobile, com foco no desenvolvimento de aplicações Android.

O objetivo é praticar e consolidar a criação de aplicativos móveis, aplicando conceitos de interface, interação com o usuário, consumo de APIs e integração com serviços externos.

💡 Todos os projetos foram desenvolvidos e testados utilizando o Android Studio como ambiente de desenvolvimento.

---

## 📚 Conteúdo
- Introdução ao sistema operacional Android  
- Estrutura de aplicações (Activities, Services, Broadcast Receivers)  
- Gerenciadores de layout (LinearLayout, ConstraintLayout, entre outros)  
- Componentes visuais (TextView, Button, EditText, ImageView, RecyclerView)  
- Manipulação de eventos do usuário  
- Navegação entre telas (Activities e Intents)  
- Configurações com AndroidManifest.xml  
- Consumo de APIs e serviços web  
- Integração com Firebase  

---

## 🎯 Conceitos abordados
- Arquitetura e funcionamento do Android  
- Orientação a objetos aplicada ao mobile  
- Desenvolvimento de interfaces responsivas  
- Comunicação com serviços externos  
- Boas práticas no desenvolvimento mobile  
- Organização e estruturação de projetos Android
